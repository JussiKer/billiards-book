# Website (Free Hosting)

This folder contains a static website for Billiards Book:
- `index.html` (landing page)
- `privacy.html` (privacy policy page)
- `styles.css`

## Quick publish to GitHub Pages

1. Push this repository to GitHub.
2. In GitHub repository settings, open **Pages**.
3. Set source to your default branch and folder `/site`.
4. Save.
5. GitHub gives you a public URL like:
   `https://<username>.github.io/<repo>/`

Use privacy policy URL in Google Play Console:
`https://<username>.github.io/<repo>/privacy.html`

## Update Google Play link

After your app is live, edit `site/index.html` and replace:
`https://play.google.com/store`
with your real app listing URL.
